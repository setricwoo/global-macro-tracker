# 全球宏观事件追踪系统

## 项目简介

这是一个专业的全球宏观事件追踪网页工具，用于跟踪2026年全球重大政治经济事件，适用于券商宏观经济分析。

## 功能特点

- **交互式世界地图**：在地图上直观展示全球事件分布
- **事件分类筛选**：支持按地缘冲突、央行会议、国际峰会、政治选举、贸易协定等类型筛选
- **时间轴视图**：按时间顺序展示近期事件
- **未来一周关注**：突出显示近期重要关注点
- **详细事件弹窗**：包含概述、时间线、前瞻、新闻链接等多标签页信息
- **国家事件模块**：在地图上直接显示各主要国家最重要的事件

## 事件类型

1. **地缘冲突** - 俄乌冲突、巴以冲突、红海危机、朝鲜半岛局势、台海局势
2. **央行会议** - 美联储、欧央行、日本央行、中国央行、英国央行等
3. **国际峰会** - G20、APEC、北约峰会、金砖峰会、中国两会等
4. **政治选举** - 美国中期选举、德国大选、韩国大选、日本参议院选举等
5. **贸易协定** - 中美经贸磋商、RCEP、CPTPP等

## 使用方法

### 本地运行

1. 直接在浏览器中打开 `index.html` 文件
2. 或使用本地服务器：
   ```bash
   python -m http.server 8080
   ```
   然后访问 http://localhost:8080

### 部署到服务器

将以下文件上传到Web服务器：
- `index.html`
- `styles.css`
- `app.js`
- `events-data.js`
